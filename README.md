# Projeto Faculdade

Este repositório contém atividades e exercícios desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas.

## Sobre o projeto

O objetivo deste projeto é praticar conceitos de programação utilizando JavaScript, aplicando os conhecimentos adquiridos em sala de aula e em atividades acadêmicas.

## Tecnologias utilizadas

- JavaScript
- Node.js
- GitHub

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

- Lógica de programação
- Variáveis e tipos de dados
- Estruturas condicionais
- Laços de repetição
- Funções
- Organização de código
